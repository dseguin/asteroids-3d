Prototype for a 3D 'Asteroids' game.

Controls:
--------
  pitch    - mouse up/down
  yaw      - mouse left/right
  roll     - Q/E
  forward  - W
  backward - S
  left     - A
  right    - D
  up       - LSHIFT
  down     - LCTRL
  shoot    - left mouse button
  toggle camera drift - BACKSPACE
  toggle debug info   - BACKTICK/TILDE
  quit     - ESC

Dependencies:
------------
  SDL    >= 2.0.1
  OpenGL >= 1.5

Build with `make release-c89`
